阅读GRE文章的正确姿势

Aug 13, 2015

阅读做题的步骤

  1. 看长阅读还是短阅读
  2. 看看题目数目
  3. 看看第一题
  4. 读文章＋7原则＋做骨架＋找中心
  5. 做题 general题目还是detail

做题的姿势

  1. 桉顺序做题
  2. 遇到难题排除
  3. 实在不行猜一个，不要留着以后做
  4. mark一下
  5. 有空回头看看

排除法的标记：

A. X
B. √
C. ~

如何正确地读

要关注结构，积极高效地读：最开始读得慢，抓住重点，后来读得快。先看题目再在原文中找，或者太抓细节都是不高效的方式。

七原则

  1. 投入：a 哄自己开心：“太好了，下面6分钟我可以了解一下海啸。”b 辨别好坏，如果海洋受到A的威胁，对A生气，这个坏家伙！
  2. 找简单故事：几个字概括
  3. 读→大脑形成图像
  4. 正在读的和刚读过的联系起来：支持还是反对，有无回答之前的问题
  5. 注意信号：分段和信号词
  6. 加快速度别陷入细节，除了：段落开头，Big Surprise, Big Result

** Point 是核心**

我：不加分辨地读全文→做题→被难题困住→读文章→随便猜一个

更好：抓住文章结构

短文章

一道题：快速读后选

3道题：理解文章

** 做笔记**

  1. 每段中心
  2. Big Suprise
  3. Big Result
  4. 读完后找全文中心

长文章

  1. 首段细细读
  2. 剩下段落找中心以及在全文中的作用，细节略去，需要时再读细节。 别读细节，还会起反作用，不仅理解不了中心，而且细节题靠记忆不靠谱，还得回文定位才好
  3. 读完找中心

题型

general questions

不用读文章，直接做题

specific questions

题 → 关键词 → 笔记 → 回文定位 → 再读 → 凝结成5个字的“真理” → 再看题与选项 → 消除和真理差的远的

策略

  1. 每个字都必须对
  2. 避免极端的选项
  3. infer不推理

时间

verbal30分钟，填空15分钟，阅读15分钟，阅读每题1.5分钟，那么如果是一道题的阅读，短时间搞定。两道题的3分钟。长文章6分钟，3分钟读文章＋笔记，做题3分钟，general：45秒，specific：60秒。
