我的 MacOS 应用

Oct 16, 2019

编辑器

  • Emacs: 免费, 最常用的编辑器了, 用 org-mode 记笔记, ess 运行 R 语言, 爱不释手. 虽然配置和学习麻烦了些, 用了好多年了, 但是每次想改动什么的时候, 还是发现自己是个新手.
  • sublime: 付费, 最常用的功能是多行同时编辑并且合并.
  • MacDown: 免费, Markdown 编辑器, 虽然 Emacs 也可以编辑 Markdown, 有时候还是想所见即所得

Menu bar 常驻

  • Hammerspoon: 免费, 功能丰富, 我用来快速启动应用和管理窗口大小. 比如使用 左边的 option 键作为 meta 键, option + m 快速启动 emacs. 缩放窗口定义为 option + f 当前活跃窗口到最大. 可以参考我的配置文件 .hammerspoon
  • Karabiner-Elements: 免费, 改键工具, 上一次 MacOS 升级到 macOS High Sierra 之后, Karabiner 不能兼容, 我这才发现这个软件对我多么重要, 只好回退了 MacOS 的版本. 作为一个 Emacs 用户, 我经常需要用到 Control 键. Karabiner-Elements 帮我把右边的 return 键, 改成当和其他键一起按动时候, 就成了 control 键. 不敢想象没有这个改键工具的 MacOS 体验.
  • 鼠须管: 免费, 虽然配置很麻烦, 还是趁手的, 中英文切换用 shift 很顺手
  • Dropbox: 付费, 同步数据
  • Alfred: 付费, 购买了 lifetime license, 应用启动器, 文件搜索器, 拼写检查器, 剪贴板管理器, 还用这个应用来强制杀死其他应用.

书影音

  • MarginNote: 付费, 有教育优惠, 读 pdf 教材的时候, 可以列出来本书结构, 并且在树状结构上做笔记, 很方便.
  • FBReader: 免费, 方便打开比如 awz, epub 和 mobi 格式的书
  • IINA: 免费, 方便打开视频和音乐文件

语言学习

  • 欧陆词典: 付费, 查单词方便一些, 自己导入词库之后, 可以查同义词, 英语解释, 例句, 可以参考奶爸1.6G Mdict词库的补充及在Bluedict中使用的心得
  • Anki: 免费, 背单词软件. 背单词一般在手机上背, 编辑单词卡片一般在电脑上完成, 需要查单词音频, 连系图片, 具体操作流程可以读书Fluent Forever 或者看网页Using Anki Online: Chapter 2 和 Fluent forever Anki.

信息源和笔记

  • Reeder: 付费, RSS 阅读器, 新闻和博客文章信息源, 虽然现在软件升级到 4 了, 我还在用着版本 3, 感觉已经够好用了
  • Evernote: 免费, 我付费了高级功能, 在稍后读并且存档 文章里提到过, 因为需要把文章通过邮件收藏到 evernote, 必须解锁高级功能.
  • Pocket: 免费, 稍后读, 把 Reeder 中的文章, 暂时没时间读的, 加入稍后读

Chrome 插件

  • Vimium: 在网页上使用 vim 键位, 常用翻页, 到页头, 页尾
  • AdGuard: 屏蔽广告
  • Evernote Web Clipper: 剪切收藏网页
  • Save to Pocket: 把文章保存到 Pocket

其他

  • OmniDisksweeper: 免费, 磁盘清理工具. 经常发现电脑硬盘又快满了, 但是又不知道到底哪些文件占据了大量空间. Mac OS 自带大文件检测功能, 但是问题是系统自带功能有限, 很多时候, 我都发现硬盘系统的文件占了太多空间, 而系统文件是不在大文件检测功能扫描范围内的. 这时候就需要 OmniDisksweeper 了, 扫描系统文件需要一些时间, 但是之后删除就方便多了
  • Macpass: 免费, 密码管理软件, 虽然 MacOS 有 Keychain 这个软件可以管理密码, 但是毕竟信不过把所有密码交给苹果公司.
  • iTerm2: 免费, 最好用的 terminal emulator
  • ocenaudio: 免费, 音频处理软件, 有时候需要把一段音频裁剪一小段放入 Anki, 背单词时候跟读, 这个软件完全能够满足我的需求.
